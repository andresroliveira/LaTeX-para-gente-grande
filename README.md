# LaTeX para gente grande

## Descrição

Este que repositório é um guia para o minicurso LaTeX de para Ensino gente de grande a ser ministrado no LEM - Laboratório de Ensido de Matemática do IMECC - Instituto de Matemática, Estatística e Computação Científica da UNICAMP - Universidade Estadual de Campinas.

Este minicurso tem como objetivo apresentar técnicas não tão básicas de L A TEX para aqueles que já possuem conhecimento básico da ferramenta e desejam aprimorar suas habilidades.

## Referências

A pasta `Referencias/` contém PDFs de apoio usados durante a preparação do minicurso.  
Ela não está versionada no Git; os arquivos podem ser baixados novamente pelos links abaixo.

### Referências gerais de LaTeX

- [The Not So Short Introduction to LaTeX2e](https://ctan.org/tex-archive/info/lshort/english/lshort.pdf)  
  Introdução geral ao LaTeX, cobrindo estrutura de documentos, comandos básicos, matemática, figuras, tabelas e bibliografia.

- [LaTeX2e unofficial reference manual](https://latexref.xyz/latex2e.pdf)  
  Manual de referência para comandos e ambientes do LaTeX2e.

- [A Short Introduction to LaTeX](https://www.techscribe.co.uk/ta/latex-introduction.pdf)  
  Introdução breve ao LaTeX, útil como material complementar.

- [LaTeX2e for authors](https://www.latex-project.org/help/documentation/usrguide.pdf)  
  Guia oficial do LaTeX Project voltado a autores de documentos.

- [LaTeX2e for class and package writers](https://www.latex-project.org/help/documentation/clsguide.pdf)  
  Guia oficial do LaTeX Project para criação de classes e pacotes.

### Matemática

- [User's Guide for the amsmath Package](https://mirrors.ctan.org/macros/latex/required/amsmath/amsldoc.pdf)  
  Manual principal do pacote `amsmath`.

- [Technical notes on the amsmath package](https://mirrors.ctan.org/macros/latex/required/amsmath/amsmath.pdf)  
  Documentação técnica do pacote `amsmath`.

- [mathtools documentation](https://mirrors.ctan.org/macros/latex/contrib/mathtools/mathtools.pdf)  
  Manual do pacote `mathtools`, extensão importante do `amsmath`.

- [Math into LaTeX](https://ctan.org/tex-archive/info/mil/mil.pdf)  
  Referência clássica para escrita matemática em LaTeX.

### Figuras, gráficos e apresentações

- [LaTeX graphics guide](https://mirrors.ctan.org/macros/latex/required/graphics/grfguide.pdf)  
  Guia oficial para inclusão e manipulação de gráficos no LaTeX.

- [Beamer User Guide](https://mirrors.ctan.org/macros/latex/contrib/beamer/doc/beameruserguide.pdf)  
  Manual oficial da classe `beamer`.

- [PGF/TikZ Manual](https://mirrors.ctan.org/graphics/pgf/base/doc/generic/pgf/pgfmanual.pdf)  
  Manual principal do PGF/TikZ.

- [A very minimal introduction to TikZ](https://cremeronline.com/LaTeX/minimaltikz.pdf)  
  Introdução curta e prática ao TikZ.
